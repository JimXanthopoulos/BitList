A personal game library web app where you can search for games and build your own list.

## [BitList](https://jimxanthopoulos.github.io/BitList/)

It uses the RAWG Video Games Database API for the list and the carousel on the front page.
I would be very happy if anyone used it as a template for their own personal project, just try and get your own personal API key from RAWG. It's FREE!!! :)
